# Workshop informatie

Dit is het start project voor het doen van de workshops. Hierin staat informatie en code voor de lessen.

## Workshop 1 - introductie & instalatie

Installeren van de volgende onderdelen <br>
**Java** 17 of nieuwer https://jdk.java.net/archive/ <br> <br>
**IntelliJ**
https://www.jetbrains.com/idea/download/#section=windows <br/><br/>
**Postman** https://www.postman.com/downloads/?utm_source=postman-home <br/><br/>
**GIT** https://git-scm.com/book/en/v2/Getting-Started-Installing-Git <br/><br/>
**Docker** https://docs.docker.com/desktop/install/windows-install/ <br/><br/>
**Jira** Maak een
account <br/> `https://id.atlassian.com/signup?continue=https%3A%2F%2Fwww.atlassian.com%2Fgateway%2Fapi%2Fstart%2Fauthredirect%3FatlOrigin%3DeyJpIjoiZmZiZDEzYmMxNjZjNGNmZWE2OGVmODUzZmJhMjIxNzgiLCJwIjoid2FjLWdsb2JhbGRyb3Bkb3duIn0` <br><br/>
**Maven** https://maven.apache.org/install.html (Deze is optioneel omdat er een integrate is met IntelliJ)

Als laatste voor deze les voer de [Main](src/main/java/week1/Main.java) en
de [MainTest](src/test/java/week1/MainTest.java) in de test folder uit.
<br> Dan ben je klaar voor Workshop 2

Mocht je willen controleren of je docker ook werkt. clone dan deze
repo `https://github.com/feddevanderlist/TestContainersDemoApplication.git` <br>
en draai de CucumberTestSuite in de src/it/java/nl/vandalist/it folder <br>
Dit zou het resultaat moeten zijn [foto](extraResources/ResultaatVanTestcontainers.png)
